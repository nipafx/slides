=== Patterns so far...

In `instanceof` and `switch`, patterns can:

* match against reference types
* deconstruct records
* nest patterns
* ignore parts of a pattern

In `switch`:

* refine the selection with guarded patterns

	System.out.println(b + " in [-128, 127]");
