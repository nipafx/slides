=== Lots to talk about!

Final features in Java 23:

* Markdown in JavaDoc
* generational ZGC by default

New preview features:

* primitive patterns
* module imports

=== Lots to talk about!

* unnamed patterns
* FFM API
* multi-source-file programs

Ongoing preview features:

* +++<s>string templates</s>+++
