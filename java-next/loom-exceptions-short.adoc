=== Exceptions

Pinning: ::
* a _pinned_ VT will block the PT
* caused by:
** +++<s>object monitors (<code><span class="hljs-keyword">synchronized</span></code>)</s>+++ ⇜ ㉔
** +++<span style="font-size: 0.5em;">class initialization</span>+++ ⇜ ㉖
** native calls

Capture: ::
* a _captured_ VT blocks the PT
* caused by file I/O
